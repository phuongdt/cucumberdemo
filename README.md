#How to run project
    Sure that mvn is installed
    Open terminal
    Clone project from github
        git clone https://github.com/phuongdt/cucumberdemo.git
    Run test
        mvn clean install
        or
        mvn test
    If want import project to eclipse
        mvn eclipse:eclipse
